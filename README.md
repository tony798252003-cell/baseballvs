# ⚾ 棒球對決

Vue 3 + Vite + Tailwind CSS 打造的互動式棒球計分應用

## 功能特色

- 🎮 單人無限打擊 / 雙人攻守交換模式
- 📊 即時記分板與壘包顯示
- ⚡ 記球數功能（好球/壞球）
- 🎵 球員加油歌播放
- 📋 CSV 球員名單匯入

## 安裝與運行

```bash
# 安裝依賴
npm install

# 啟動開發伺服器
npm run dev

# 打包生產版本
npm run build

# 預覽生產版本
npm run preview
```

## 技術棧

- Vue 3 (Composition API)
- Vite
- Tailwind CSS
- JavaScript
